Web y móvil

La mayor parte de nuestro tiempo está dedicado al diseño y desarrollo web, más aún ahora que la web es accesible desde una infinidad de dispositivos. El desafío es grande, las tecnologías cambian todos los días y no hay respuestas definitivas. Analizamos cada caso y aplicamos lo mejor para cada proyecto. Hemos explorado muchas metodologías de trabajo, tecnologías y estilos de diseño.

Interactivos Punto de Venta (PoP)

El contexto lo es todo, a veces una gran pantalla táctil con una aplicación para los presentes es mucho más efectivo que miles de avisos de prensa. Concursos, interactivos, sorteos, ruletas, interacciones cortas y efectivas para eventos y ubicaciones específicas.
